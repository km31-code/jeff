# Jeff v1 Build Plan

## Product Direction

By Phase 10, Jeff has full foundational coworker capabilities from Phases 1-9.  
Phase 10 focuses on **interaction coherence** so users talk to Jeff instead of operating many panels.

## Completed Foundations

- Phase 0: desktop scaffold
- Phase 1: task/workspace persistence
- Phase 2: ingestion/retrieval
- Phase 3: shell + grounded chat
- Phase 4: voice + interruption
- Phase 5: proactive coworking runtime
- Phase 6: safe revision lifecycle
- Phase 7: bounded parallel subtasks
- Phase 8: flow-aware suggestions
- Phase 9: end-to-end coherence + observability hardening

## Phase 10 Focus

- companion-first default UI
- intent routing from normal conversation into existing systems
- inline conversational action controls
- full workspace/details hidden by default but available on demand

## Constraints

- no backend feature expansion
- no removal of existing core systems
- no browsing/multi-task/autonomy-scope growth

## Phase 10 Exit Criteria

- app opens into simple companion view
- user can start interacting immediately
- revision/subtask/suggestion actions are triggerable from normal conversation
- panel-heavy workspace remains available through explicit toggle
- underlying systems continue to pass regression tests
